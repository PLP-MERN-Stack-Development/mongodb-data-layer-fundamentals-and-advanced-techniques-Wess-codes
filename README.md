# PLP Bookstore MongoDB Assignment

## How to Run
1. Install Node.js and MongoDB driver (`npm install mongodb`).
2. Update `insert_books.js` with your Atlas connection string.
3. Run `node insert_books.js` to populate the database.
4. Use `queries.js` in MongoDB Shell (`mongosh`) or Compass to test queries.

## Files
- `insert_books.js`: Script to insert sample books.
- `queries.js`: Contains all CRUD, advanced queries, aggregation, and indexing tasks.
- Screenshot: Shows `plp_bookstore` database and `books` collection in Atlas.
